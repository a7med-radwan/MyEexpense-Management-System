# مشروع مصروفاتي (My Expenses) 

نظام متكامل لإدارة المصروفات الشخصية مبني باستخدام إطار عمل Laravel، يساعدك على تتبع دخلك ونفقاتك وإنشاء خطط مالية ذكية.

##  المميزات الرئيسية

- **لوحة تحكم تفاعلية (Dashboard):** تعرض ملخصاً سريعاً للوضع المالي.
- **إدارة الدخل والمصروفات:**
    - إضافة، تعديل، وحذف مصادر الدخل.
    - إضافة، تعديل، وحذف المصروفات وتصنيفها.
- **نظام الفئات (Categories):** تنظيم المصروفات ضمن فئات مخصصة لسهولة التفصيل.
- **التخطيط المالي (Plans):** إنشاء خطط مالية للمستقبل (مع تكامل ذكي لتوليد الخطط).
- **نظام مصادقة (Authentication):** تسجيل دخول وتسجيل مستخدمين جدد بشكل آمن.
- **واجهة مستخدم عصرية:** تصميم متجاوب وأنيق باستخدام Tailwind CSS.

##  التقنيات المستخدمة

- **Backend:** [Laravel 12](https://laravel.com) - أحدث إصدار من إطار عمل PHP القوي.
- **Frontend:** [Tailwind CSS 4](https://tailwindcss.com) - لتصميم الواجهات.
- **Database:** MySQL / SQLite.
- **Tools:** Vite (Build tool), Composer (PHP Dependency Manager), NPM (JS Package Manager).
